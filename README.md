# Анализатор киберугроз

Скрипт анализирует события сетевой безопасности на основе логов Suricata и данных API Vulners.

Функции:
1. Загрузка логов Suricata
1. Анализ IP адресов.
2. Анализ типов атак.
3. Получение данных об уязвимостях через API Vulners.
4. Обнаружение подозрительных IP.
5. Имитация блокировки IP.
6. Сохранение отчётов (JSON, CSV).
7. построение графиков (PNG).

Используемые технологии:
Python, pandas, matplotlib, requests.

Источники данных:
1. Suricata logs.
2. Vulners API.

Запуск:
python analyzer.py
